Common issues:
• "No course table found" — The sections table must be visible before you click the bookmarklet. Search for a course first.
• Course name shows as UNKNOWN — The sections still import correctly. This just means the dropdown text couldn't be detected automatically.
• Alert never appears — Your browser may block bookmarklets. Use the "Can't drag?" manual option in Step 1 to add it a different way.
• "Could not read clipboard" — Click inside the text box above and press Ctrl+V (Windows) or Cmd+V (Mac) to paste manually.
